Document and Website Structure

HTML allows you to mark up sections of content based on their meaning and structure. In addition to defining individual parts of your page (such as "a paragraph" or "an image"), HTML also boasts a number of block level elements used to define areas of your website (such as "the header", "the navigation menu", "the main content column").

Take some time to plan a basic website structure, and write the HTML to represent this structure. Basic sections of a document include the header, navigation, main content, and footer.

Note: The appearance of the page shown here - including the layout, colors, and fonts - is achieved by applying CSS to the HTML. Once you have an understanding of the basics of HTML, try diving into our CSS first steps module to start learning how to style your site.

HTML Basics

HTML is a standard markup language for web page creation. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) such as tags and attributes. HTML has a lot of use cases, namely:

  • Creating static web pages
  • Creating email templates
  • Creating web applications

It's also worth noting that HTML is not considered a programming language as it can't create dynamic functionality. It is now considered an official web standard. The World Wide Web Consortium (W3C) maintains and develops HTML specifications, along with providing regular updates.

HTML Tags

An HTML opening tag is enclosed by a pair of angle brackets in the form of <tag-name> (e.g., <p>, <title>), which is associated with a matching closing tag </tag-name> having a leading forward slash, (e.g., </p>, </title>). The tag-name shall be in lowercase.

Attributes, in the form of name="value" pairs can be included in the opening tag to provide additional information about the element. For example, in <html lang="en">, the attribute lang="en" specifies the language of the document.

You can use a <link> element to add an external CSS Style Sheet (and <style>...</style> element for internal styles): <link href="filename.css" rel="stylesheet">

You can use a <script>...</script> element to define programming scripts. For example, to add an external JavaScript file: <script src="filename.js"></script>
